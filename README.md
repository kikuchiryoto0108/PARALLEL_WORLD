こちらは一年生の際、c++のコンソールアプリにて制作した2d/3dを切り替えて遊ぶ、新感覚パズルアクションゲームです。
ルールは、ホットスター★（星）を5つ集めて、ゴール前の氷を溶かし、ゴールすることが条件です。

// ************************
// 操作方法
// ************************
ゲーム共通
Ｍ：2D/3D切り替え
Enter：決定

2Dの場合
A：左移動
D：右移動
SPACE：ジャンプ
Ｓ：土管に入る（土管の上で）

3Dの場合
WASD：移動
SPACE：ジャンプ
SHIFT：土管に入る（土管の上で）
